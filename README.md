# Dynamic Programming In Elixir

A simple dynamic programming example in Elixir.

## Install the dependencies
```shell
$ mix deps.get
```

## Run the tests

```shell
$ mix test
```

[Here.](https://github.com/danieljaouen/dynamic-programming-in-lfe)
